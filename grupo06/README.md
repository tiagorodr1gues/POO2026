# MindStep 🧠
O foco da aplicação web tem como objetivo incluir jovens com THDA, desenvolvida no âmbito da UC POO

## Estrutura MVC Simplificada

project/
├── html/
│   └── index.html          # Página principal
├── css/
│   └── styles.css
├── js/
│   ├── model/
│   │   └── taskModel.js    # Dados e lógica
│   │   ├── taskView.js     # Views das tarefas
│   │   └── commonView.js   # Views partilhadas (notificações)
│   └── init.js             # Injeção de dados
├── img/
│   └── media/
└── README.md

## Como usar o projeto!
Abrir o `html/index.html` num browser.
